---
title: 039_SETLINE
---

-   Opcode: **0x039**
-   Short name: **SETLINE**
-   Long name: Set Line Bounds

#### Argument

none

#### Stack

  
*X1*

*Y1*

*Z1*

*X2*

*Y2*

*Z2*

**SETLINE**

#### Description

Sets the bounds of this line object (for its touchOn, touchOff, and across scripts). Lines are actually 3d hitboxes, not lines.
