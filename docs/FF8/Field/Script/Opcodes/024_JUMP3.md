-   Opcode: **0x024**
-   Short name: **JUMP3**
-   Long name: Jump to 3-coordinate location

#### Argument

Walkmesh triangle ID.

#### Stack

  
*XCoord*

*YCoord*

*ZCoord*

*Speed in frames?*

**JUMP3**

#### Description

Make this entity jump to the given coordinates. I'm 99% positive the fourth parameter is the speed of the jump in frames.
