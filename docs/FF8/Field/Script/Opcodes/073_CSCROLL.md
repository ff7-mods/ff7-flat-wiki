---
title: 073_CSCROLL
---

-   Opcode: **0x073**
-   Short name: **CSCROLL**
-   Long name: C? Scroll

#### Argument

none

#### Stack

  
*X*

*Y*

*Frame count*

**LSCROLL**

#### Description

Scrolls the screen by X and Y pixels over *frame count*. Parameters are untested, but I'd be very surprised if they're wrong.
