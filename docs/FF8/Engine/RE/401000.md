**Custom name**: SetFF8Path

**Description**: Builds and sets root FF8 ('\\\\ff8\\') path. Do not mistake with [0x004011C0](4011C0.md)!

------------------------------------------------------------------------

**Arguments**:

char\[\] string - Root path string. Example: *""\\\\ff8""*
