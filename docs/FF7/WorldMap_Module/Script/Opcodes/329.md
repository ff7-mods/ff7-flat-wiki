- Opcode: **0x329**
- Name: set camera tilt speed
- Two-word opcode: No

#### Parameters

| Location | Name  |           Description           |
|:--------:|:-----:|:-------------------------------:|
|  Stack   | Speed | How fast should the camera tilt |

#### Notes

Only works when camera zoom/tilt is enabled using opcode [0x31c](31c).
