---
title: 32a
---

- Opcode: **0x32a**
- Name: set camera zoom speed
- Two-word opcode: No

#### Parameters

| Location | Name  |              Description               |
|:--------:|:-----:|:--------------------------------------:|
|  Stack   | Speed | How fast should the camera zoom in/out |

#### Notes

Only works when camera zoom/tilt is enabled using opcode [0x31c](31c).
